# Login Page

A modern, responsive login/signup page built with React and Vite. Features multiple theme options and a personalized user dashboard.

## Tech Stack

- React 18 with Vite
- React Router for navigation
- Tailwind CSS for styling
- Lucide React for icons

## Features

- User registration with form validation
- Multiple color themes (Purple, Red, Green, Orange, Dark Mode)
- Password visibility toggle
- Personalized welcome dashboard
- Profile editing capabilities
- Fully responsive design
