# Login Page

A modern login and registration page built with React and Vite. Includes form validation, dark/light mode toggle, and user dashboard.

## Technology Stack

- React 18
- Vite
- React Router
- Framer Motion
- Tailwind CSS
- Lucide React

## Features

- User registration with real-time validation
- Password strength indicator
- Dark and light mode toggle
- Page transition animations
- Toast notifications
- Loading skeleton screens
- Glassmorphism UI design
- Input focus glow effects
- 3D card hover animations
- Profile editing
- Fully responsive layout

## Setup

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd Login-Page-1
```

2. Install dependencies:
```bash
npm install
```

### Development

Start the development server:
```bash
npm run dev
```

The application will open at `http://localhost:5173` (or another port if 5173 is in use).

### Build

Create a production build:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```
